# Nova Yapı İnşaat - Kurumsal Web Sitesi

Modern, responsive ve kullanıcı dostu bir inşaat firması tanıtım sitesi.

## 🚀 Kullanılan Teknolojiler
- **React.js** (Vite ile)
- **Tailwind CSS** (Hızlı ve modern tasarım)
- **Framer Motion** (Akıcı animasyonlar ve etkileşim)

## 🛠 Özellikler
- **Single Page Application (SPA):** Hızlı ve kesintisiz gezinti.
- **Dinamik Filtreleme:** Projeleri kategorilere göre sıralama.
- **Lightbox:** Proje detaylarını incelemek için modern bir görüntüleyici.
- **İletişim:** Hızlı WhatsApp entegrasyonu.

*Bu proje, staj kapsamında yazılım mühendisliği yetkinliklerini geliştirmek amacıyla hazırlanmıştır.*
